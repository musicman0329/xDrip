Choose your CGM!
================

xDrip+ supports the following CGM data sources:
  * xBridge Wixel
  * Wifi Wixel / Parakeet
  * Wifi Wixel + BT Wixel
  * Wifi Wixel + xBridge Wixel
  * G4 Share Receiver
  * G5 Transmitter
  * Libre Bluetooth
  * xDrip+ Sync Follower
  * LibreAlarm
  * 640G / EverSense
  * Disable Collection
